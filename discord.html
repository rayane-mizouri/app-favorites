Pourquoi utilisé discord



Discord est un logiciel de messagerie instantanée.
Il fonctionne sur les systèmes d’exploitation:
Windows, macOS, Linux, Android, iOS ainsi que sur les navigateurs web.
Ce qui fait donc de discord la messagerie instantanée révée.



Discord est né en 2015 son créateru est Jason Citron,
il créer cette application afin de rassembler tous les logiciels de VoIP existants
(Skype, TeamSpeak, Mumble, etc.) dans un seul logiciel.
Le logiciel est d'abord utilisé par les joueurs,
mais d'autres communautés se mettent rapidement à l'utiliser, comme les développeurs.
